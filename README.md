Install dependecies

```shell
Yarn
```

Run app

```shell
Yarn start
```

Run Emulador IOS in MAC

```shell
cd ios && pod install
```
Back to root folder

```shell
cd ..
```

```shell
react-native run-ios --simulator "iPhone 11 Pro Max"
```

Test app

```shell
Yarn test
```
